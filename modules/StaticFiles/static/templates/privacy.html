AIDE and its developers do not, and never will, collect or use your personal data.

In the context of the workings of this service, the following data is being stored, resp. collected.

Data stored on server

AIDE stores the following data in plain text in a relational database on the server (restricted and password-protected access by default):
  • Your user account information (name and E-mail address)
  • For every viewed image and provided annotation: the user agent information of your Web browser, the browser's window size in pixels and the timestamp and duration at, resp. during which you viewed the image(s) and created the annotation(s)

Cookies used

AIDE sets the following cookies in your browser:
  • Your AIDE username (in plain text)
  • A session token, encrypted with a user-specific secret token (both randomly generated); HttpOnly
  • Boolean flags (whether to suppress cookie message, whether to skip the tutorial, whether to skip the annotation confirmation dialog, whether burst mode is enabled)
  • Values of AI model prediction and conversion thresholds per project
  • Any cookies set by third-party libraries, such as jQuery

What AIDE does not do

By default, AIDE will never do the following:
  • Store your IP address anywhere
  • Employ any form of tracking service, especially third-party
  • Provide any outside party with any of the user data
However, the developers behind AIDE cannot prevent others using the system of breaking these promises. The developers of AIDE take no responsibility for any such action. If in doubt, please contact the system administrator responsible for this specific installation of AIDE.
